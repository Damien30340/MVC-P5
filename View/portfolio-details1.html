Déscription du projet

Dans le cadre de ma formation "Développeur d'application", je devais trouver une solution web pour le client à partir d'un CMS. Mon choix pour ce site s'est porté sur Wordpress avec comme thème Ocean-WP couplé à Elementor. Le client qui arrive sur le site de locations/ventes peut s'il le souhaite, reserver directement en ligne via un plugin. Les disponibilités sont affichés en direct.

Liste compétences acquises à l'issu de ce projet :

  • Concevoir un projet Wordpress à partir d'un cahier des charges
  • Installation Wordpress, installation bdd, configuration d'un thème
  • Ajouts et modifications de pages, plugins
  • Création d'un accès compte administrateur
  • Deploiement, soutenance orale auprès du client fictif
